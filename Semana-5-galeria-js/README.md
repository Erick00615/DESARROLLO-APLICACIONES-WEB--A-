# Galería Interactiva con JavaScript

Proyecto desarrollado con HTML, CSS y JavaScript que permite agregar imágenes mediante URL, seleccionarlas y eliminarlas dinámicamente.

## Funcionalidades
- Agregar imágenes usando JavaScript y DOM.
- Seleccionar una imagen a la vez.
- Eliminar la imagen seleccionada.
- Diseño responsive con CSS Grid.
